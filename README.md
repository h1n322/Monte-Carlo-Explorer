Програмний комплекс для моделювання методом Монте-Карло

Ця програма розроблена в рамках моєї науково-дослідницької роботи для МАН (секція "Комп'ютерна інженерія").

Головна мета проєкту — створити інструмент, який дозволяє на практиці застосувати метод Монте-Карло для оцінки фінансових ризиків та прогнозування поведінки акцій. На відміну від звичайних прогнозів, ця програма показує не одну ціну, а діапазон ймовірностей (конус невизначеності).

Функціонал програми

Програма має кілька режимів роботи:

Прогноз акцій (модель GBM):

Будує тисячі можливих сценаріїв руху ціни на основі Геометричного броунівського руху.

Автоматично завантажує реальні котирування з Yahoo Finance (за тикерами, наприклад AAPL, BTC-USD).

Має захист від збоїв: якщо немає інтернету, автоматично переходить на демо-дані, щоб можна було показати роботу алгоритму.

Backtesting (Перевірка на історії):

Дозволяє "повернутися в минуле": навчити модель на даних до певної дати (наприклад, до 2022 року) і порівняти прогноз із тим, що сталося насправді у 2023 році.

Оцінка Європейських опціонів:

Розраховує теоретичну вартість Call-опціону, використовуючи симуляцію Монте-Карло.

Математична перевірка:

Режими обчислення числа $\pi$ та визначеного інтеграла. Я додав їх для верифікації генератора випадкових чисел — щоб довести, що базовий алгоритм працює коректно перед тим, як довіряти йому фінансові розрахунки.

Структура проєкту

Я використав модульну архітектуру, щоб код був чистим і структурованим:

App.py — головний файл запуску (інтерфейс, потоки).

Stocks.py — модуль завантаження даних та прогнозування акцій.

Options.py — модуль фінансової математики (опціони).

Integral.py та Pi.py — математичні ядра для тестування методу.

Як запустити

Варіант 1: Запуск .exe файлу

У папці dist (або в релізах) знаходиться файл MonteCarloApp.exe. Його можна запустити на будь-якому комп'ютері з Windows, встановлювати Python не потрібно.

Варіант 2: Запуск з коду

Якщо ви хочете подивитися код або запустити скрипт вручну:

Встановіть необхідні бібліотеки:

pip install -r requirements.txt


Запустіть головний файл:

python App.py
